# Sales Performance Dashboard 📊

## Overview
This project focuses on analyzing sales data to uncover key business insights and support data-driven decision-making. Using Power BI, an interactive dashboard was developed to visualize sales performance across different regions, products, and time periods.

## Objective
- Identify top-performing regions and products  
- Analyze sales trends and patterns  
- Support business decisions with data-driven insights  

## Key Insights
- Morocco identified as the top-performing region based on sales analysis  
- Clear variations in sales performance across regions and product categories  
- Opportunities for growth in underperforming markets  

## Tools & Technologies
- Power BI – Data visualization and dashboard development  
- Excel / CSV – Data source and preprocessing  

## Features
- Interactive dashboard with filters and slicers  
- Regional sales comparison  
- Product performance analysis  
- Time-based sales trends  

## Dashboard Highlights
- Visual representation of sales by region  
- KPIs for total sales and performance tracking  
- Easy-to-use interface for business insights  

## How to Use
1. Open the `.pbix` file using Power BI Desktop  
2. Explore the dashboard using filters and slicers  
3. Analyze insights based on region, product, and time  


## Outcomes
- Improved understanding of regional sales performance  
- Identified high-performing and underperforming areas  
- Enabled data-driven business decisions
